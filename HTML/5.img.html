Image tag is img

By this tag we add image to our web page. also remember that this is a SELF CLOSING TAG.

Attributes in img tag are:

src: this is required and it need the value of image source.
alt:this is also another very useful attribute. it is mostly used as text for screen readers and also if the image is not loaded correctly.
