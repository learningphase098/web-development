ANCHOR ELEMENT

anchor element helps in creating hyperlink in different areas like to a web page , email even telephone etc.
there are different attribute in anchor tag also ..

let me explain tha attributes

An attribute is just a additional information about a tag. They are always included in a opening tag. There can be no attribute in a tag and also more than one attribute. Each attribute has a value which is specified by using an equal to sign. and attributes are seperated by just a backspace.

THIS IS HOW ATTRIBUTES AND TAG LOOK LIKE:
<TAG > ATTRIBUTE1="VALUE" ATTRIBUTE2="VALUE"> TEXT </TAG>

Now for ANCHOR TAG there are many attributes

  • href : this is for the url of the link you want to attach.
    • Telephone numbers with tel: URLs
    • Email addresses with mailto: URLs
    • SMS text messages with sms: URLs
  • target:This is just ask where to display.
    • self: The current browsing context. (Default)
    • blank: Usually a new tab, but users can configure browsers to open a new window instead.
